SpyWatch

Third party tracking on the web, through cookies and other means, is one of the most widespread forms of user data collection. Since much can be inferred about a user by mining the pages he/she has visited (i.e. interests, gender, purchases, trips, location, etc), third parties can construct profiles of users and gather sensitive personal information. The surreptitious nature of this tracking, often occurring without the user's consent or knowledge, is alarming. SpyWatch is a Google Chrome extension which allows users to see the third party sites tracking them online. You can download SpyWatch here!

Automatically Generating Custom Search Engines

From September 2012 to 2013, I worked on developing algorithms that would automatically create a search engine for a given context. Users often struggle with keyword based search engines like Google or Bing because queries can have multiple interpretations and search engines fail to understand the context in which the user is looking for information. For example, while a doctor and patient may issue similar queries, they are both looking for very different types of information. My work was based on the intuition that the Google Custom Search Engine (CSE) infrastructure could be leveraged to create unique search engines for different user contexts (doctors, patients, etc). I developed algorithms which, utilizing information about the user’s context, scraped the web and processed/filtered candidate sites that could be used to create a CSE. I used the algorithm to create the following search engines :

  • A medical CSE for doctors at Stanford Hosptial
  • A US history CSE for history students at Gunn High School.
