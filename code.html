Code

Code for specific papers is available on the publications page. The following are open source projects/resources:

  • simple-wikidata-db: a simple library for triplifying the Wikidata dump and extracting triples.
  • legal-ml-datasets: a collection of pointers to datasets at the intersection of machine learning and law.
  • legal-segmenter: a very simple segmentation library for legal texts.
